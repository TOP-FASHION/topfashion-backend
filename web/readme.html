Installazione: La famosa installazione in 5 minuti

  1. Decomprimi in pacchetto in una directory vuota e carica tutto sul server remoto.
  2. Lancia dal browser la pagina /wp-admin/install.php. Sarai portato alla pagina che ti aiuterà a impostare il file wp-config.php con i dettagli di connessione del tuo database.
    1. Se per qualche motivo non dovesse funzionare, non preoccuparti. Non funziona con tutti gli hosting web. Apri wp-config-sample.php con un editor di testi come WordPad o similari e inserisci i dettagli di connessione del tuo database.
    2. Salva il file come wp-config.php e caricalo sul server.
    3. Apri wp-admin/install.php nel tuo browser.
  3. Non appena il file di configurazione è configurato, il processo di installazione creerà le tabelle necessarie per il tuo sito. Se c’è qualche errore, controlla due volte il file wp-config.php, e provaci ancora. Se fallisce ancora, chiedi nei forum di supporto fornendo tutte le informazioni che hai a tua disposizione.
  4. Se non inserisci una password, prendi nota della password che ti viene assegnata. Se non scegli un nome utente, sarà admin.
  5. Il processo d’installazione dovrebbe rimandarti alla pagina di login. Collegati con il nome utente e la password che hai scelto durante l’installazione. Se la password è stata generata per te, puoi fare doppio clic su “Profilo” per modificarla.

Aggiornamento

Aggiornamento automatico

Se stai aggiornando dalla versione 2.7 o successive, puoi utilizzare l’aggiornamento automatico:

  1. Apri wp-admin/update-core.php nel tuo browser e segui le istruzioni.
  2. Pensavi ci fosse di più? Questo è tutto!

Aggiornamento manuale

  1. Prima di aggiornare l’installazione, assicurati di aver fatto una copia di backup di tutti i file che hai modificato, come ad esempio il file index.php.
  2. Cancella i vecchi file di WP, salvando quelli che hai modificato.
  3. Carica i nuovi file sul server.
  4. Dirigi il browser alla pagina /wp-admin/upgrade.php.

Aggiornamento da altre piattaforme

WordPress può importare contenuti da altre piattaforme. È necessario che abbia installato e configurato WordPress secondo le istruzioni precedenti, prima di utilizzare i nostri strumenti d’importazione.

Requisiti di sistema

  • PHP versione 5.2.4 o superiore.
  • MySQL versione 5.0 o superiore.

Raccomandati

  • PHP versione 5.6 o superiore.
  • MySQL versione 5.6 o superiore.
  • Il modulo Apache mod_rewrite.
  • Un link a wordpress.org nel tuo sito.

Risorse online

Se hai delle domande alle quali questo documento non ha dato risposta, puoi approfittare delle numerose risorse online di WordPress:

Il progetto WordPress Codex
Il «Codex» è l’enciclopedia su tutto quello che riguarda WordPress. È la risorsa di informazioni più completa su WordPress.
Il blog di sviluppo
Qui puoi trovare gli ultimi aggiornamenti e le ultime notizie su WordPress. Aggiungi il sito ai segnalibri e visitalo spesso.
WordPress Planet
«WordPress Planet» è un aggregatore di notizie che riporta gli articoli riguardanti i blog su WordPress in rete.
I forum di supporto di WordPress
Se hai cercato in rete e non hai trovato risposta, i forum di supporto sono molto attivi e costituiscono una comunità molto ampia di persone pronte ad aiutarti. Per fare in modo che possano aiutarti, assicurati di scrivere dei titoli molto descrittivi e riporta le tue domande nella maniera più dettagliata possibile.
Il canale IRC di WordPress
Infine, c’è un canale chat dove le persone si incontrano per chiacchierare di WordPress e per occasionali richieste di aiuto. La pagina wiki sopracitata dovrebbe mostrarti la giusta direzione (irc.freenode.net #wordpress).

Note finali

  • Se hai suggerimenti, idee, commenti o se (gasp!) scopri un bug, vieni a trovarci nei Forum di supporto
  • WordPress ora dispone di un robusto plugin API che semplifica l’estensione del codice. Se sei uno sviluppatore e sei interessato ad utilizzare questo plugin, consulta la documentazione del plugin sul Codex. Nella maggior parte dei casi, non dovrai modificare niente del codice principale.
