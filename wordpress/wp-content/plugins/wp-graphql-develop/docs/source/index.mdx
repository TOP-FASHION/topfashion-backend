**WPGraphQL** brings the power of GraphQL to your WordPress site.

This is the documentation site for WPGraphQL, the free,
open-source WordPress plugin that brings the power of GraphQL to
WordPress.

For more info about WPGraphQL check out our <Link to="/guides/about-wpgraphql/">WPGraphQL Guide</Link> or continue to <Link to="/getting-started/install-and-activate/">Install and Activate</Link>.
